## [Unreleased]

### Added
- Initial release
- PHP 8.5 pipe operator support
- Functional validation with error accumulation
- ValidationContext (functor) for error collection
- Reusable validators (IntegerValue, StringValue)
- Union types for type-safe error handling
- MultipleValidationContext for entity validation
- ErrorsBag and MultipleFieldErrorsBag for error collection
- Example Value Objects (Age, Name, Email, Password, Person)
- Comprehensive test suite using Pest PHP
